\# Forest Fire Prediction ML Project



This is an end-to-end machine learning project that predicts forest fire risk based on environmental features.



\## Features

\- Data preprocessing and feature engineering

\- Model training using Ridge Regression

\- Flask web application for predictions



\## Tech Stack

\- Python

\- Flask

\- Scikit-learn

\- Pandas



\## How to Run

1\. Run the app:

&#x20;  python application.py
